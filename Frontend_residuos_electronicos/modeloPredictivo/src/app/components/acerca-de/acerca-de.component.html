Modelo predictivo de residuos electrónicos en Guayaquil

Acerca de Datos estadísticos actuales Modelo de predicción por año Modelo de predicción por filtros

Este sistema utiliza un modelo de red neuronal para realizar predicciones sobre la cantidad de residuos electrónicos generados, en la ciudad de Guayaquil. El modelo toma en cuenta factores como el nivel educativo, ingresos, y tipos de dispositivos desechados, junto con tasas de crecimiento específicas para realizar estimaciones futuras. Al entrenar el modelo con datos de una muestra de 387 personas y proyectar el crecimiento de los residuos, se obtiene una predicción de toneladas de residuos electrónicos que se generarán en un periodo determinado.
